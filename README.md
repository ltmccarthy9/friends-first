# friends-first - (development)

## Description
Friends First is the best application for making friends and forming 
relationships.
 

## Technologies Used

* React & React Router
* MongoDB & Mongoose
* Nodejs
* Express
* Tailwindcss
* Firebase
* Redux/Redux Toolkit
